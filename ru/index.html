Ключевые особенности¶

УЧЕТ И ПЛАНИРОВАНИЕ

Программа содержит все необходимые инструменты для детального финансового учета. Например, поддерживается многовалютность, сплиты, периодические операции. В каждом отчете можно увидеть операции, из которых сложилась та или иная цифра.

КОНФИДЕНЦИАЛЬНОСТЬ

Без рекламы. Без отслеживающих трекеров. Без регистрации. Доступ к программе ограничен при помощи графического ключа. Резервные копии и сообщения используемые в коллективной работе шифруются.

0 трекеров, см. Exodus Privacy Project

ИМПОРТ СМС И PUSH УВЕДОМЛЕНИЙ

Программа умеет автоматически создавать операции из СМС и Push уведомлений, определять статью, проект, персону, получателя и плательщика. Поддерживается 200+ банков. Если Вашего банка не окажется в списке, мы его добавим.

КОЛЛЕКТИВНАЯ РАБОТА

Возможна полная синхронизация между устройствами, совместный финансовый учет только по выбранным счетам, проектам, персонам, контрагентам, статьям, или сбор данных на одном устройстве, в случае, когда, скажем, родители отслеживают расходы детей. Регистрации на сайте разработчика не нужна.

ДОСТУП С КОМПЬЮТЕРА

Вы можете сформировать отчеты на любом устройстве, компьютере, ноутбуке, планшете. Причем без регистрации и доступа к сайту разработчика.

ПОДДЕРЖКА

Вы можете задать любой вопрос на форуме или по электронной почте. Доступно руководство пользователя.

Отличия между версиями BStore и Google Play¶

Версии приложения для Google Play и BStore отличаются:

  1. Google Play не разрешает получать список установленных пакетов. Поэтому, чтобы ограничить список источников push-уведомлений или выбрать получателя событий API, необходимо указать список пакетов вручную. В версии для BStore пакеты можно выбрать из списка.

  2. Google Play не разрешает использовать специальные возможности (Accessibility service). Поэтому для чтения уведомлений доступен только один способ через доступ к уведомлениям. В версии для BStore для чтения уведомлений доступно два способа, специальные возможности и доступ к уведомлениям.
