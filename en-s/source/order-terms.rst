Payment, delivery and return policy
===================================

Payment for ordering a non-exclusive license to use the application is made through the payment service https://robokassa.com.

Delivery of a non-exclusive license to use the application in the form of an order number is made instantly, immediately after payment. The order number is displayed in the browser window and duplicated in the electronic receipt. The order number is used when downloading and receiving application updates.

A full refund is available within 7 days of the purchase of a non-exclusive license to use the application. To return, send an email to |contact_email| with a description of the reason for return and the order number.

If you have any questions, please contact |contact_email|
