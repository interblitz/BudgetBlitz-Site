Key features
------------

.. grid:: 2
  :gutter: 1

  .. grid-item-card:: :material-regular:`credit_card;3em;sd-text-success` DEEP FINANCIAL ACCOUNTING

      The app contains all necessary tools for a detailed financial accounting. It supports Dollars, Euros, Pounds, and all other currencies in the world. You can use splits, reccuring transactions. Drilldown report and chart are usefull to analyze spendings and revenues.

  .. grid-item-card:: :material-regular:`security;3em;sd-text-success` PRIVACY

    No ads. No trackers. No registration. You can restrict app access using graphical lock pattern. Backups and teamwork messages are encrypted.

    :bdg-link-primary-line:`Trackers free, see on Exodus Privacy Project <https://reports.exodus-privacy.eu.org/en/reports/biz.interblitz.budgetfree/latest/>`

  .. grid-item-card:: :material-regular:`sms;3em;sd-text-success` SMS AND PUSH NOTIFICATIONS SUPPORT

    The app can create transactions from SMS or Push notifications. Category, project, person, payee and location can be assigned automatically when importing transactions.

  .. grid-item-card:: :material-regular:`sync;3em;sd-text-success` TEAMWORK

    Full devices transactions syncing, or only selected accounts, projects, persons, payees, categories transactions syncing, or gathering all transactions from different devices in a single one supported. No registration needed at all.

  .. grid-item-card:: :material-regular:`devices;3em;sd-text-success` PC ACCESS

    Make reports on any device you like, PC, notebook or tablet PC. No registration and developers site access required.

  .. grid-item-card:: :material-regular:`help;3em;sd-text-success` SUPPORT

    You can get support on our discussion forum or by email. The app contains context help and user manual of course.
